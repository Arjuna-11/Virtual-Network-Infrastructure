# Virtual-Network-Infrastructure
tested a virtual network infrastructure for connectivity and load

Technologies: Virtual PC Simulator (VPCS), Networking, Routing, Linux, TCP/IP

This project focused on configuring and testing virtual network infrastructure environments using simulated routers, PCs, and TCP/IP networking concepts. The environment involved configuring IP addressing, subnet masks, gateways, and routing communication between multiple virtual systems to simulate segmented network architectures. Network connectivity and routing functionality were validated using ICMP ping testing and command-line networking utilities while troubleshooting addressing and gateway configuration issues within isolated virtual environments. The project provided hands-on experience with network infrastructure fundamentals, routing concepts, subnet configuration, and enterprise-style network communication workflows.
